## 项目概述
本项目旨在开发一套基于嵌入式AI的智慧校园人员检测系统，通过集成物联网技术、边缘计算与轻量化AI模型，实现校园内人员动态的实时监测与管理。系统硬件由ESP32-CAM摄像头模块和树莓派5处理终端构成，采用YOLO等轻量级AI算法进行本地化数据处理，仅上传检测结果至云平台，确保数据隐私安全。前端展示平台基于Vue.js框架开发，支持多终端访问，可实时查看各场景人员分布、历史数据分析及异常告警。该系统适用于图书馆、教室、食堂等高需求场所，通过优化资源调度、预警异常事件，提升校园管理效率和安全性，为智慧校园建设提供创新技术支持。
